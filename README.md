# Carlos Martin — 12-Week Communication Curriculum

An interactive flipbook built for Carlos Martin, a drilling engineer working on the Anad Peru project. This is his personal 12-week English communication curriculum, built by Better Life English.

---

## How to open it

Download `carlos-flipbook.html` and open it in any browser. No internet connection required after the file loads. No login. No app.

---

## How to navigate

- **Arrow buttons** at the bottom of the screen
- **Keyboard arrow keys** — left and right
- **Swipe** left or right on mobile

---

## Built with

- HTML / CSS / Vanilla JavaScript
- Google Fonts — Space Grotesk, Cormorant Garamond, Inter
- No frameworks. No dependencies. No build step. Single file.

---

## Notes

- The file is self-contained. Everything is inside the one HTML file.
- To edit content, open the file in any code editor and find the page by its comment — for example `<!-- PAGE 5: PHASE 1 -->`.
- To add a new page, duplicate an existing page block, update the `id` to the next number, and update `const TOTAL` in the script at the bottom.
